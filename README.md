# 真核細胞構造與功能配對

這是一個專為高中生物課程設計的互動式教育應用程式。透過 2.5D 互動模型，讓學生能夠直觀地探索真核細胞（包含動物細胞與植物細胞）的內部構造，並透過配對遊戲加深對各個胞器功能的記憶。

## ✨ 核心功能

* **2.5D 互動式細胞模型**：使用精緻的 SVG 與流暢的動畫效果，呈現細胞內部構造。
* **動 / 植物細胞切換**：學生可以自由切換動物細胞與植物細胞，藉此觀察兩者在構造上的差異（例如：植物特有的葉綠體與細胞壁，動物特有的中心粒與溶體）。
* **功能配對挑戰**：點選畫面中的胞器後，在右側選單中找出對應的生理功能進行配對。
* **即時視覺回饋**：
  * 滑鼠懸停與點選時的發光效果。
  * 配對成功時的綠色高光與標籤顯示。
  * 配對失敗時的紅色震動提示。
* **學習進度追蹤**：畫面上方設有進度條，完成所有 12 個構造的配對後，會顯示恭喜畫面並提供重新開始的選項。

## 🔬 包含的細胞構造

本應用程式涵蓋了 12 種重要的真核細胞構造：

1. **細胞核 (Nucleus)** - 具有遺傳物質，可控制細胞生理活動
2. **內質網 (Endoplasmic Reticulum)** - 修飾、運輸蛋白質與脂質
3. **核糖體 (Ribosome)** - 合成蛋白質
4. **粒線體 (Mitochondria)** - 進行呼吸作用產生能量的場所（細胞發電廠）
5. **高基氏體 (Golgi Apparatus)** - 儲存、運輸、修飾及分泌物質
6. **細胞膜 (Cell Membrane)** - 區隔細胞內外的環境，控制物質進出
7. **細胞質 (Cytoplasm)** - 進行化學反應的場所
8. **液泡 (Vacuole)** - 儲存水及代謝廢物
9. **溶體 (Lysosome)** - 分解細胞內的大分子物質及衰老胞器 *(動物細胞限定)*
10. **中心粒 (Centriole)** - 細胞分裂時可協助染色體分離 *(動物細胞限定)*
11. **葉綠體 (Chloroplast)** - 進行光合作用的場所 *(植物細胞限定)*
12. **細胞壁 (Cell Wall)** - 保護、支持及維持細胞的形狀 *(植物細胞限定)*

## 🛠️ 技術棧 (Tech Stack)

* **前端框架**: React 19 + Vite
* **樣式設計**: Tailwind CSS v4
* **動畫引擎**: Framer Motion (`motion/react`)
* **圖示庫**: Lucide React
* **圖形渲染**: 互動式 SVG (Scalable Vector Graphics)

## 🚀 本地開發與運行

1. 安裝依賴套件：
   ```bash
   npm install
   ```

2. 啟動開發伺服器：
   ```bash
   npm run dev
   ```

3. 建置正式環境版本：
   ```bash
   npm run build
   ```

## 🎮 操作說明

1. 觀察畫面中央的細胞模型。
2. 點擊右上角的「動物細胞」或「植物細胞」按鈕來切換細胞類型。
3. 點選模型中發光或感興趣的胞器（例如：細胞核）。
4. 在右側的「功能配對清單」中，點擊你認為正確的功能描述。
5. 若配對正確，清單項目會亮起綠燈並標示胞器名稱；若錯誤則會震動提示。
6. 將所有 12 個胞器全部配對完成即可通關！
